# MAX7219 Plugin для YoRadio (параллельный дисплей)

## Что это
Плагин добавляет второй дисплей на MAX7219 (матрицы 8x8) и работает параллельно с основным дисплеем YoRadio.

Показывает:
- Станция и трек
- Погода (встроенная YoRadio)
- Время / дату
- IP
- Громкость при изменении
- Номер станции при смене
- В STOPPED: коротко "STOP", затем цикл Время → Погода → IP → Дата

## Требования
- ESP32/ESP32-S3
- Библиотеки:
  - MD_Parola (majicDesigns)
  - MD_MAX72XX (majicDesigns)
- Файл шрифта: `fonts/5bite_rus.h`

## Подключение (Software SPI)
MAX7219 -> ESP32-S3
- VCC -> 5V
- GND -> GND
- DIN -> GPIO 40
- CLK -> GPIO 41
- CS  -> GPIO 39

Если будут глюки по сигналам (редко, но бывает):
- уменьшить `MX_INTENSITY`,
- укоротить провода,
- использовать буфер 74AHCT125/74HCT (уровни 3.3V->5V).

## Установка (Arduino IDE)
1) Установи библиотеки:
   Скетч → Подключить библиотеку → Управлять библиотеками:
   - MD_Parola
   - MD_MAX72XX

2) Скопируй файл `MAX7219_plugin.ino` в папку проекта YoRadio (рядом с `YoRadio.ino`).

3) Убедись, что есть `fonts/5bite_rus.h`.
   Если папки `fonts/` нет — создай её и положи туда файл шрифта.

4) Открой `YoRadio.ino` в Arduino IDE и прошей как обычно.

## Настройка
Все настройки находятся в начале `MAX7219_plugin.ino`.

Главные:
- `MX_MAX_DEVICES` — количество матриц (4..16).
- `MX_CLK_PIN / MX_DATA_PIN / MX_CS_PIN` — пины подключения.
- `MX_INTENSITY` — яркость 0..15.
- `MX_SCROLL_SPEED` — скорость прокрутки (меньше = быстрее).
- блоки `MX_SHOW_...` — включение/отключение информации.

## Погода (встроенная YoRadio)
Плагин НЕ делает отдельные запросы к погоде.
Он показывает строку `timekeeper.weatherBuf`, которую заполняет YoRadio.

Чтобы погода появилась:
- включи погоду в настройках YoRadio (showweather),
- задай ключ/город (как требует твоя сборка),
- дождись синхронизации времени (NTP) и первого обновления погоды.

## Важно (если не компилируется)
В проекте не должно быть двух файлов, которые объявляют одни и те же callbacks:
- `network_on_connect()`
- `player_on_stop_play()`
- `player_on_start_play()`

Если у тебя уже есть другой плагин с такими функциями — оставь только один.
