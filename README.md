# YoRadio MAX7219 Display Extension (parallel display)

Плагин добавляет **второй дисплей на матрицах MAX7219 (8x8, FC-16)** для YoRadio.
Работает **параллельно** с основным дисплеем (OLED/TFT) и не требует правки исходников YoRadio.

## Возможности
- Бегущая строка: **станция + трек**
- **Встроенная погода YoRadio** (берётся из `timekeeper.weatherBuf`)
- **Время / дата**
- **IP-адрес**
- Показ **громкости при изменении**
- Показ **номера станции при переключении**
- **Умный пропуск**: если погоды нет — блок погоды пропускается без ожидания
- Режим **STOPPED**: коротко показывает `STOP`, затем цикл:
  `Время (центр) → Погода → IP → Дата`

## Совместимость
Проверено на:
- ESP32-S3
- Arduino-ESP32 core 2.0.17
- YoRadio v0.9.720
- 8 модулей MAX7219 FC-16

## Зависимости (Arduino Library Manager)
Установить через: *Sketch → Include Library → Manage Libraries…*
- **MD_Parola** (majicDesigns)
- **MD_MAX72XX** (majicDesigns)

## Подключение (Software SPI)
MAX7219 → ESP32-S3:
- VCC → 5V
- GND → GND
- DIN → GPIO 40
- CLK → GPIO 41
- CS  → GPIO 39

Если появляются артефакты/срывы:
- уменьши `MX_INTENSITY`,
- укороти провода,
- при необходимости используй буфер 74AHCT125/74HCT (уровни 3.3V→5V).

## Установка в YoRadio
1) Скопируй файл `MAX7219_plugin.ino` в папку проекта YoRadio **рядом с `YoRadio.ino`**.
2) Скопируй папку `fonts/` в корень YoRadio (чтобы был путь `YoRadio/fonts/5bite_rus.h`).
                                                                                        5bite_rus.h -данный файл создан и свободно распространяется его Автором bill-gilbert,
файл взят из открытого источника: https://4pda.to/forum/index.php?showtopic=1010378&view=findpost&p=119069902
4) Открой `YoRadio.ino` в Arduino IDE и прошей как обычно.

⚠️ В проекте YoRadio не должно быть второго файла/плагина, который объявляет те же callbacks:
- `network_on_connect()`
- `player_on_stop_play()`
- `player_on_start_play()`

Если такие уже есть в другом плагине — оставь только один.

## Настройка
Все настройки находятся в начале `MAX7219_plugin.ino`:
- `MX_MAX_DEVICES` — число матриц 8x8 (4..16)
- `MX_CLK_PIN / MX_DATA_PIN / MX_CS_PIN` — пины
- `MX_INTENSITY` — яркость 0..15
- `MX_SCROLL_SPEED` — скорость (меньше = быстрее)
- `MX_SHOW_...` — включение/отключение блоков

## Погода (встроенная YoRadio)
Плагин не делает отдельные запросы в интернет.
Он показывает строку `timekeeper.weatherBuf`, которую формирует сам YoRadio.

Чтобы погода появилась:
- включи погоду в настройках YoRadio (`showweather`)
- укажи ключ/город (как требует твоя сборка)
- дождись синхронизации времени (NTP) и первого обновления погоды

Примечание: в YoRadio символ `*` в погоде часто используется вместо `°`.
Плагин автоматически заменяет `*` на значок градусов (код 247 в шрифте).
